Realiza una búsqueda por texto.

public buscar(PDO $conexion, string $texto) : array<string|int, mixed>

Localiza libros cuya información coincida parcialmente con el texto indicado. Se buscan coincidencias en el título del libro, nombre y apellidos del autor y en la nacionalidad.

Parameters
$conexion : PDO

Conexión a la base de datos.

$texto : string

Texto a buscar dentro de los diferentes campos.

Return values
array<string|int, mixed> —

Lista de resultados coincidentes, cada uno con datos de libro y autor.
